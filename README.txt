# Study App Neon PWA

Archivos para convertir tu proyecto en una Progressive Web App (PWA).

## Incluye:
- manifest.json
- icon-192.png
- icon-512.png
- snippet para index.html

## Qué hacer:
1. Sube todos estos archivos a GitHub.
2. Agrega el contenido de head-snippet.html dentro del <head> de index.html
3. Espera unos minutos.
4. Vuelve a PWABuilder.
